## V1.1 — 관리자 대시보드 고도화

> **상태**: 완료 (2026-07-07)

### 목표
견적서 관리를 더 효율적으로 할 수 있도록 관리자 전용 레이아웃과 UX를 개선합니다.

### 기능 1: 관리자 레이아웃 견적서 목록

**배경**: 현재 견적서 목록(`/dashboard/quotes`)은 기본 대시보드 레이아웃을 공유합니다. 견적서 관리에 특화된 레이아웃과 뷰가 필요합니다.

**요구사항**:
- 견적서 목록을 테이블 형태로 표시 (견적서 번호, 클라이언트, 발행일, 유효기한, 금액, 상태)
- 상태별 필터링 (전체 / 작성중 / 발송완료 / 승인 / 반려 / 만료)
- 각 행 클릭 시 견적서 상세 페이지로 이동

**인수 기준**:
- 견적서 목록이 관리자 전용 레이아웃에서 렌더링됨
- 상태 필터 선택 시 해당 상태 견적서만 표시
- 로딩 중 Skeleton UI 표시

---

### 기능 2: 견적서 목록에서 공유 링크 복사

**배경**: 현재 공유 링크 복사는 견적서 상세 페이지(`/dashboard/quotes/[id]`)에서만 가능합니다. 목록 페이지에서 바로 링크를 복사할 수 있으면 워크플로우가 단축됩니다.

**요구사항**:
- 견적서 목록 각 행에 "링크 복사" 버튼 추가
- 공유 링크가 없는 경우: 클릭 시 자동 생성 후 클립보드 복사
- 공유 링크가 있는 경우: 바로 클립보드 복사
- 복사 완료 시 sonner 토스트 알림

**인수 기준**:
- 목록에서 링크 복사 버튼 클릭 시 `POST /api/quotes/[id]/share` 호출 후 URL 클립보드 복사
- 이미 공유 중인 경우 API 호출 없이 즉시 복사
- 복사 성공/실패 토스트 표시

**구현 포인트**:
```
QuotesTable.tsx
  └─ 각 행에 "링크 복사" 아이콘 버튼 추가 (Link2 아이콘)
  └─ 공유 중 여부: quote.shareSlug && quote.isPublic 판별
  └─ useCopyToClipboard() 훅 재사용
  └─ 공유 미생성 시: POST /api/quotes/[id]/share → 복사
  └─ 공유 생성 중: loading 상태로 버튼 비활성화
```
